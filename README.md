# Thought Loop - לולאת המחשבה

## תיאור כללי

**Thought Loop** הוא משחק פאזל-פלטפורמר 2D שבו השחקן משתמש במכניקה ייחודית של לולאות זמן. בכל לולאה של 10 שניות, השחקן מבצע פעולות שמשוחזרות על ידי "שכפול רוח-רפאים" בלולאה הבאה. השחקן צריך לתאם בין מספר גרסאות של עצמו כדי לפתור חידות מורכבות - ללחוץ על מתגים בו-זמנית, להעביר חפצים, ולהגיע ליעדים שאי-אפשר להגיע אליהם לבד.  
**ז'אנר:** Puzzle Platformer, Time Manipulation  
**קהל יעד:** גילאי 12+, חובבי משחקי חשיבה כמו Portal, Braid, The Witness
